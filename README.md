# Yovisto UI

## Installation

```bash
$ npm install
```

## Build And Run

```bash
# build the project
$ npm run build
# build the image (if needed) and start the container, listening on port 3010
$ docker-compose up --build
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode (need `npm run build` first)
$ npm run start:prod
```
