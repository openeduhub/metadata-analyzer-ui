# Yovisto UI

## Installation

```bash
$ npm install
$ npm run generate
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Docker

```bash
# build the docker image
$ npm run docker-build 
# start container, listening on port 3010
$ docker-compose up
```
